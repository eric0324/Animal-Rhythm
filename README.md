# 節奏挑戰

一款簡單的節奏遊戲。

## 功能

- 4 x 2 格子的圖片網格
- 圖片會依照節奏依序閃爍
- 背景音樂同步播放
- 自動循環多組題目卡牌

## 使用方式

1. 使用瀏覽器開啟 `animal-rhythm-game.html`
2. 點擊畫面任意處開始遊戲
3. 跟著音樂節奏觀察亮起的圖片

## 專案結構

```
.
├── animal-rhythm-game.html   # 主程式
├── bg.mp3                    # 背景音樂
├── images/
│   └── questions/            # 題目圖片
│       ├── 1/                # 第 1 組 (1.png ~ 8.png)
│       ├── 2/                # 第 2 組
│       ├── ...
│       └── 11/               # 第 11 組
└── README.md
```

## 自訂題目

在 `images/questions/` 目錄下新增資料夾，放入 8 張 PNG 圖片（命名為 `1.png` ~ `8.png`），並在 HTML 中的 `decks` 陣列加入對應編號即可。
